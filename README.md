# Warehouse Distribution System

Цей проект -- система розподілу товарів між складами та торговими точками, реалізована на ASP.NET Core Razor Pages (.NET 9).

## Основний функціонал

- **Облік складів та товарів:** Відображення залишків по кожному складу з деталізацією по категоріях та одиницях виміру.
- **Розподіл ресурсів:** Автоматичний розподіл товарів по торгових точках з урахуванням потреб кожної точки та наявності на складі.
- **Аналіз розподілу:** Відображення статусу забезпечення кожної точки по кожному товару.
- **Інтеграція з LLM:** Можливість використовувати зовнішній AI-сервіс для генерації підказок або аналізу (налаштовується через `appsettings.json`).

## Як запустити проект
 
 1. Запустіть локальну бд trans.sql. Ви можете використовувати будь-який інструмент для роботи з PostgreSQL, наприклад, pgAdmin або командний рядок psql.


2. створіть файл з назвою "appsettings.json" у кореневій папці вашого проекту і додайте туди:

{
    "ConnectionStrings": {
        "DefaultConnection": "Host=localhost;Database=назваБд;Username=postgres;Password=пароль"
    },
    "LLM": {
        "ApiKey": "YOUR_API_key",
        "Endpoint": "https://api.groq.com/openai/v1/chat/completions"
    },
    "Logging": {
        "LogLevel": {
            "Default": "Information",
            "Microsoft.AspNetCore": "Warning"
        }
    },
    "AllowedHosts": "*"
}

3. Запустіть проект через run або натиснувши F5, він відкриється у браузері.
